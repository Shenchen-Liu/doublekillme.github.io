草 / 赋得古原草送别

发表于 2019-02-15 | 分类于 唐代
字数统计: | 阅读时长 ≈

作者：白居易
离离原上草，一岁一枯荣。 野火烧不尽，春风吹又生。 远芳侵古道，晴翠接荒城。 又送王孙去，萋萋满别情。

荔枝香 吴道贤携赠蜜荔枝

发表于 2019-02-15 | 分类于 清代
字数统计: | 阅读时长 ≈

作者：董元恺
新岁松关高卧。谁相过。吾友满袖烟霞，携饷炎州果。 倾来白玉明肌，露滴瑶浆破。奚必黛叶、湘枝绛囊裹。 想五月，星缀珠排如火。凤爪龙牙，津液酸甜颗颗。 陈紫遗芳，色味经时隔江左。好待熏风鼓柁。

菊坡崔参政说平叛卒不得已拜经略之命岂敢言

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：戴复古
角巾私第自逍遥，诸老之中此老高。 无可柰何怀印绶，甚非得已用弓力。 风生玉帐千兵肃，天落金牌一札褒。 缓急经心护乡井，生憎儿辈说功劳。

荷池撑伞作

发表于 2019-02-15 | 分类于 近现代
字数统计: | 阅读时长 ≈

作者：卢青山
雨中来双燕，适与我相邻。斜掠或蘸水，高飞乃入云。 清荷疏未满，临风泻玉银。宛若群姬舞，裾袂翻翩纷。 忽尔凝神立，其思若幽人。新者犹未起，水面点圆清。 旧者忽已颓，折弃下青萍。青萍漠不识，流去何轻盈。 有鸟所罕见，来向此中停。上立亭盖软，下踞小叶沉。 俄顷自南北，点水惊蜻蜓。路畔多行客，曾不一回身。 落落伫立久，不觉心沉凝。嗟彼雨荷兮，何自发于斯，移向山涧里，独秀或相宁。

菩提偈

发表于 2019-02-15 | 分类于 唐代
字数统计: | 阅读时长 ≈

作者：惠能
菩提本无树，明镜亦非台。 佛性常清净，何处有尘埃！ 身是菩提树，心为明镜台。 明镜本清净，何处染尘埃！ 菩提本无树，明镜亦非台。 本来无一物，何处惹尘埃！ 菩提只向心觅，何劳向外求玄？ 听说依此修行，西方只在目前！

菩萨蛮 其三 庚午重阳前二日青岛海滨晚步

发表于 2019-02-15 | 分类于 近现代
字数统计: | 阅读时长 ≈

作者：梁启勋
参差楼阁淩云起。疏星摇曳空蒙里。灯火出空栊。帘垂灯影红。 行人回懒步。谁解馀心素。海气动轻寒。归来兴已阑。

菩萨蛮 拟古四首次伯孝韵 其二

发表于 2019-02-15 | 分类于 近现代
字数统计: | 阅读时长 ≈

作者：朱庸斋
隔帘蜂蝶曾相识。拘红管翠劳春力。掩面怯相逢。屏山何处通。 旧情拚委弃。空剩还珠泪。烟水望中遥。新妆人正娇。

菩萨蛮·书江西造口壁

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：辛弃疾
郁孤台下清江水，中间多少行人泪。西北望长安，可怜无数山。 青山遮不住，毕竟东流去。江晚正愁余，山深闻鹧鸪。 (愁余 一作：愁予)
