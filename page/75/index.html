感怀

发表于 2019-02-23 | 分类于 清代
字数统计: 60 | 阅读时长 ≈ 1

作者：贵成

柴门镇掩剑频磨，岁月惊心电影过。拟结名流今日少，欲吟好句古人多。

疗贫无计空愁绝，作事常乖奈命何。不信读书偏误我，几回搔首发悲歌。


屠亚卿索次周司徒分献星辰二坛韵

字数统计: 60 | 阅读时长 ≈ 1

作者：林光

邕邕天乐协和銮，袅袅心香肃礼坛。珠斗艳藏苍玉斝，露华清润紫云冠。

三斟秬鬯神应格，九奏登歌夜未阑。君动臣随皆至礼，天人何以不交欢。


八月三日游方广洞天留题纪实望仙台与洞天相对乃前山也

发表于 2019-02-23 | 分类于 明代
字数统计: 114 | 阅读时长 ≈ 1

作者：郑潜

循源入苍山，乘风蹑云栈。石门境方幽，洞天路疑断。

杰阁撑岩空，危阑倚云半。络石藤为罗，攀崖竹垂缆。

龙树春长荣，羊谷迹未漫。冰帘挂晴昼，飞步惊童丱。

天瓢注龙尾，足以苏旱暵。我来当清秋，况得游览伴。

篮舆念民力，铁杖契禅观。平生泉石兴，庶儗毛骨换。

会登望仙台，松巢翠华幔。

与朱元思书

发表于 2019-02-23 | 分类于 南北朝
字数统计: 148 | 阅读时长 ≈ 1

作者：吴均

　　风烟俱净，天山共色。从流飘荡，任意东西。自富阳至桐庐一百许里，奇山异水，天下独绝。

　　水皆缥碧，千丈见底。游鱼细石，直视无碍。急湍甚箭，猛浪若奔。

　　夹岸高山，皆生寒树，负势竞上，互相轩邈，争高直指，千百成峰。泉水激石，泠泠作响；好鸟相鸣，嘤嘤成韵。蝉则千转不穷，猿则百叫无绝。鸢飞戾天者，望峰息心；经纶世务者，窥谷忘反。横柯上蔽，在昼犹昏；疏条交映，有时见日。


得程翰林扬州消息

发表于 2019-02-23 | 分类于 元代
字数统计: 165 | 阅读时长 ≈ 1

作者：揭傒斯

皇明属休期，侧席伫贞良。游麟既集薮，鸣凤亦在冈。

君子乘时兴，出饯何辉光。车马塞江介，笳鼓迎羽觞。

时秋风雨交，四顾野茫茫。晨钲激中流，飞旆忽异方。

蓄思遵旋轨，浮云纷北翔。道路閒邦畿，草木陨寒霜。

晷影倏易流，悲来但徊徨。君怀经纶忧，予负疢疾妨。

搴沈固异趣，情谊结中肠。居下众所夷，在贱谁不忘。

宛缅广陵翰，胡独置我傍。郁悒增愁思，投袂睇莽苍。

自顾匪茅茹，委质任空墙。渊渊贤达心，悢悢情内伤。

作者：乌斯道

太守携壶看藕花，倪家池馆似仙家。佳人行酒杯频覆，童子催诗鼓数挝。

山亭柳

发表于 2019-02-23 | 分类于 金朝
字数统计: 85 | 阅读时长 ≈ 1

作者：王哲

性本慈悲。要觅玄机。妙理上欲寻思。一身无可脱，被家缘、火院驱驰。

爱欲猛捐样下，怎标向、那妻儿。此个丹成分两处，若教行坐总相随。

但恐不由伊。双全全举得，人人尽、学取无为。清静到头各就，只庞居士同知。
