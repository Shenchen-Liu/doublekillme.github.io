初春雪霁游妩姥赠谢萧晴川先生

发表于 2019-02-15 | 分类于 明代
字数统计: | 阅读时长 ≈

作者：胡直
危峦仙阁临霄霭，绝壁丹梯蹑晚霞。雪霁千峰回黛色，春还万树落天花。 坐看北斗依南斗，行踏金华指玉华。愿假慈光分半席，应将文史载三车。

卜算子·我住长江头

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：李之仪
我住长江头，君住长江尾。日日思君不见君，共饮长江水。 此水几时休，此恨何时已。只愿君心似我心，定不负相思意。

卜算子·咏梅

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：陆游
驿外断桥边，寂寞开无主。已是黄昏独自愁，更著风和雨。 无意苦争春，一任群芳妒。零落成泥碾作尘，只有香如故。

卜算子·送鲍浩然之浙东

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：王观
水是眼波横，山是眉峰聚。欲问行人去那边？眉眼盈盈处。 才始送春归，又送君归去。若到江南赶上春，千万和春住。

卜算子·黄州定慧院寓居作

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：苏轼
缺月挂疏桐，漏断人初静。时见幽人独往来，缥缈孤鸿影。(时见 一作：谁见) 惊起却回头，有恨无人省。拣尽寒枝不肯栖，寂寞沙洲冷。

作者：高心夔
宠冠亲贤料遽衰，致身胡取亟登危。将军清静归醇酒，公子声华误绣丝。 坊乐入筵天庆节，殿材营第水衡司。十年风谊亏忠告，江海堙流此泪垂。

常甫宪副钦之翰检与予先后登岳钦之独值雨是日予与常甫坐候云开堂因赋二章 其一

发表于 2019-02-15 | 分类于 明代
字数统计: | 阅读时长 ≈

作者：严嵩
祠谒朝景晖，峰登云物变。今朝山中客，零雨罢登践。 林深崖路窄，幽赏不可遍。磴滑危难步，谷暗近莫辨。 兹山号灵幻，顷刻殊隐见。叨余上晴峦，万状悉输献。 愧乏摛毫赋，琢句被崖藓。君侯擅风骚，胸中富溪巘。 佳辞出天巧，不复劳洗鍊。岩峦閟辉彩，草树迷葱茜。 岂被山灵妒，或遭造物谴。并觏欣兹奇，独往惜巳殿。 期君嗣芒屐，言保箸力健。

作者：陶安
一片归心趁鹤翎，江豚忽起恣吹鯹。篷边山色和云看，枕上风声带浪听。 万事疾徐难逆料，千艘驰逐却争停。潇潇夜半无情雨，忍使寒欺短鬓零。
