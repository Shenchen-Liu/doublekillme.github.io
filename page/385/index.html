将去乡校诸生摄影留念为题一诗

发表于 2016-02-23 | 分类于 近现代
字数统计: 115 | 阅读时长 ≈ 1

作者：郭风惠

多情莫如佛，桑下不三宿。惟恐结缘深，反滋离别苦。

我昔赋归欤，乡校阒无主。妄欲裁狂狷，敢惜进愚鲁。

试携两袖风，化作一堂雨。弦诵日以安，黾勉百发举。

忽欲接淅行，危疑非乐土。立擎怀中婴，付与他人乳。

莘莘尔何知，万事今非古。念我展画图，认取翛翛羽。

刮目看阿蒙，别矣吾无语。

再次前韵

发表于 2016-02-23 | 分类于 宋代
字数统计: 204 | 阅读时长 ≈ 1

作者：朱翌

日走瞿昙宅，殷勤问觉圆。荷珠翻雨碎，荇带逆风牵。

安步穿芒屩，长驱断蜡鞭。临深呼属玉，骑气道蜿蜒。

前辈几人到，高情五字篇。翩翩遗浊世，落落度长年。

断石崖蜂出，乔枝谷鸟迁。端为逃暑去，复作拂云眠。

心旷风林豁，书乾棐几鲜。丞方有公事，尉亦从群仙。

倚玉吾何幸，栖云志未偏。久知情澹泊，不梦翠连娟。

晚日斜吹绮，新诗欲斗妍。微行窥沼鹄，孤坐蜕枝蝉。

俯视千章木，如临尺五天。韵高巾垫角，饮痛盏垂莲。

小筑坚前约，兹游定昔缘。欲誇山耸髻，安得思流泉。

却恐樽无醁，群忧爨不烟。谁来助清兴，吹裂笛如椽。


弃妇吟

发表于 2016-02-23 | 分类于 清代
字数统计: 185 | 阅读时长 ≈ 1

作者：魏锡曾

女子惮远适，临别常依依。忆昔妾来此，掩泪牵母衣。

朝别父与母，夕见姑与舅。区区一寸心，生死为君妇。

采桑起执筐，洗手归作羹。三岁靡室劳，归宁且未遑。

归宁且未遑，嗣君适远道。晨昏贱妾侍，颇谓妇窈窕。

吁嗟泰山阿，长松萦女萝。吁嗟彼何人，中乃施斧柯。

东家有贤女，鸳鸯新结侣。人生实有命，伤余何妒女。

木落无还枝，妇弃无还期。但祝舅姑寿，妾请从此辞。

念君隔两地，思君落双泪。妾去君心伤，愿君遣情累。

褰裳登长途，回首见小姑。小姑哽不语，无力回嫂车。


大宁山由云梯上塔访翠峰禅师

发表于 2016-02-23 | 分类于 明代
字数统计: 60 | 阅读时长 ≈ 1

作者：傅圭

涧草岩花迥不同，他山何处避高风。层峦如出青云外，片石疑来白雪中。

下院有松孤鹤伴，上方无路一梯通。飘飘已觉吾身健，谁复蓬莱问海东。


崇寿宫梁昭明太子修行之地

发表于 2016-02-23 | 分类于 宋代
字数统计: 65 | 阅读时长 ≈ 1

作者：李之仪

鸿禧已不凡，崇寿气尤古。经始天监谋，馀力保大举。

十围人何在，乌鹊尚如睹。井泉翳荆棘，像设半尘土。

视今犹视昔，一笑才仰俯。有愧所坐峰，安得参鹤羽。

归安学署访端木鹤田 其一

字数统计: 45 | 阅读时长 ≈ 1

作者：吴衡照
