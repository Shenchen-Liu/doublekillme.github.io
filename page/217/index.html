悼亡妻香祖诗二十首 其四

发表于 2018-02-23 | 分类于 近现代
字数统计: 164 | 阅读时长 ≈ 1

作者：吴虞

深山多豺虎，中原多盗贼。猛虎犹可防，大盗能移国。

人权重宣言，斯理世所识。自由当保障，固弗限南北。

奈何文奸言，哇声而紫色。萧墙起干戈，萁豆煎不息。

骊山土忽焦，昆池灰尽黑。幸福竟如斯，会见人相食。

当年苦专制，小民尚能活。今尊为主人，仓皇弃家室。

流离数百万，厥罪犯何律。嗟君欲逃死，古寺忍饥渴。

痴女夜半啼，忍泪不敢落。暑湿中羸躯，一病遂到骨。

伟人兴正高，志士心弥热。谁知破屋中，冤魂暗呜咽。


寄兴 其四

发表于 2018-02-23 | 分类于 清代
字数统计: 61 | 阅读时长 ≈ 1

作者：黄文仪

看破红尘淡外求，把将万事付东流。升沈有分情何著，俯仰无惭乐更优。

茗椀薰炉供静趣，行云野鹤伴閒游。一丝不挂神超越，作德由来心日休。


夜宿龙南山居闻梵

发表于 2018-02-23 | 分类于 明代
字数统计: 60 | 阅读时长 ≈ 1

作者：徐渭

寒林禅室数灯悬，清梵才闻客未眠。去远依然归夜静，来迟犹自恋香烟。

性灵微触经衣上，僧散馀鸣绕壁间。听满未方何所碍，修行谁證耳根圆。


为张四太学题画寄少司成范公

发表于 2018-02-23 | 分类于 明代
字数统计: 61 | 阅读时长 ≈ 1

作者：王世贞

故人自许同张范，手写新图托赠君。蕙帐可仍天目色，绛帷还似石头云。

矶边罢钓丝纶在，掌上挥毫雨露分。倘问吴兴王大令，郡斋亲识练为裙。


望庐山

字数统计: 45 | 阅读时长 ≈ 1

作者：赵公豫

庐山何峻极，一望尽苍清。不辨松杉色，谁闻钟磬声。

飞崖悬瀑布，幽谷语春莺。堪羡岩栖士，空馀人世情。

晓行即景

发表于 2018-02-23 | 分类于 清代
字数统计: 88 | 阅读时长 ≈ 1

作者：弘历

两山中间夹溪川，山寒岚重溪生烟。烟岚相逼黯如雾，虽雺不湿汗漫然。

策骑如游乾闼利，祥麟威凤纷腾鶱。三珠双树不根峙，层楼阿阁无地悬。

须臾金翼丽高照，真空幻有消无缘。世间万事不异此，前人后议诚谁贤。


上泸帅梁公

发表于 2018-02-23 | 分类于 宋代
字数统计: 45 | 阅读时长 ≈ 1

作者：陈天麟
旧已元枢幕，今犹大将牙。
恩威伸六诏，封域总三巴。
早晚看鸣佩，轩翔入判花。
不妨乘驷马，上冢一还家。

晓望

作者：周权
