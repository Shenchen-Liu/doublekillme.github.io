同永叔子聪游嵩山赋十二题 其十 三醉石

发表于 2019-02-15 | 分类于 宋代
字数统计: | 阅读时长 ≈

作者：梅尧臣
相期物外游，共醉仙坛石。举手薄高穹，清风生两腋。 都忘尘世烦，笑傲聊为适。

同苗发慈恩寺避暑

发表于 2019-02-15 | 分类于 唐代
字数统计: | 阅读时长 ≈

作者：李端
追凉寻宝刹，畏日望璇题。卧草同鸳侣，临池似虎溪。 树闲人迹外，山晚鸟行西。若问无心法，莲花隔淤泥。

向湖边 佘春帆消夏观荷图

发表于 2019-02-15 | 分类于 清代
字数统计: | 阅读时长 ≈

作者：姚燮
鸥渚深菰，莺栏疏柳，飒飒暗生秋意。好挈茶铛，泛扁舟来寄。 有许多、凉碧霏衫，明红流靥，写出水妍山媚。花外尘喧，任炎官停辔。 一拂烟绡，飐过蜻蜓翅。曾复拓楚簟，邀纤蟾看醉。 玉韵珠情，抵笙娱纨侍。幸豪车热马都相避。容留个、京兆风流挥麈对。 满肺酥香，沁玉田词气。

周参政惠书唁及亡儿开二首

字数统计: | 阅读时长 ≈

作者：李石
颡马非才具，蜗牛惜弊庐。 风时甘伏枕，风雨阻来车。 药裹衣巾内，梅花酒盏疏。 江湖慰寂寞，一纸故人书。

吴山秋望呈张侍御

发表于 2019-02-15 | 分类于 明代
字数统计: | 阅读时长 ≈

作者：玉芝和尚
子胥庙前江日晡，越王城头云有无。 故宫台榭几黄叶，南度寝园今绿芜。 长笛关山瞻北固，清樽鱼鸟傍西湖。 萧萧落木苍郊外，几处寒烟破屋孤。

听香斋感兴

发表于 2019-02-15 | 分类于 近现代
字数统计: | 阅读时长 ≈

作者：金鉴才
登楼望故国，人世几飘萍。雨脚临窗住，林光逼眼青。 气平何用壮，香妙只须听。莫谓黔天小，由来聚客星。

发表于 2019-02-15 | 分类于 明代
字数统计: | 阅读时长 ≈

作者：孙继皋
惜别仍为卜夜欢，春郊那问漏声残。身辞凤阙分符竹，家傍渔矶忆钓竿。 层阁勒铭他日事，横厓叱驭古人看。薄游不少青云业，倚醉无歌蜀道难。
