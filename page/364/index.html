咏秋兰

发表于 2016-02-23 | 分类于 清代
字数统计: 60 | 阅读时长 ≈ 1

作者：静诺
长林众草入秋荒，独有幽姿逗晚香。
每向风前堪寄傲，几因霜后欲留芳。
名流赏鉴还堪佩，空谷知音品自扬。
一种孤怀千古在，湘江词赋奏清商。

偈倾一百六十九首

发表于 2016-02-23 | 分类于 宋代
字数统计: 35 | 阅读时长 ≈ 1

作者：释智朋
毗蓝园裹，活中觅死不可得。
双林树下，死中觅活不可得。
脑后一锥，是活是死。

奉陪中丞苏公游招宝山同作次壁间韵二首

发表于 2016-02-23 | 分类于 明代
字数统计: 60 | 阅读时长 ≈ 1

作者：邓渼
危峰茎擢四无傍，日月东西际沆茫。
树羽孤城当昼静，楼船三翼俨云翔。
直教河水羞称伯，空笑山城僣号王。
东望三韩犹咫尺，弯弓直拟挂扶桑。

临江仙 丁酉秋窗雨夕，寄马湛翁灵隐禅林

发表于 2016-02-23 | 分类于 近现代
字数统计: 65 | 阅读时长 ≈ 1

作者：龙榆生

喜得西窗连夜雨，新凉涤尽烦襟。可消三毒一沈吟。

薰香施戒忍，弹指去来今。

最是今生难自赎，宝珠沦海深深。光风霁月企遥岑。

怎将青兕意，乞取紫阳针。


先期登高观菊岩新傍

发表于 2016-02-23 | 分类于 宋代
字数统计: 61 | 阅读时长 ≈ 1

作者：冯伯规
先得重阳五日来，清江晚日与人谐。
共搜磊磊落落句，要敌奇奇怪怪崖。
乌帽几容罗此影，黄花每喜近吾侪。
山城酒熟浑无客，一笑临风寄老怀。

拟古

发表于 2016-02-23 | 分类于 宋代
字数统计: 75 | 阅读时长 ≈ 1

作者：周端朝

秾桃复秾李，照影落春水。渡头看青杨，剪剪从风起。

裁书写连环，殷勤问行李。一笼幺凤翼，一系青鸾尾。

江南三月天，群英烂如绮。开到芜菁花，春事良未已。

何当缓缓归，荡漾绿波里。


临江仙 其六 有赠

发表于 2016-02-23 | 分类于 近现代
字数统计: 63 | 阅读时长 ≈ 1

作者：陈永正

谣诼纷纷安足数，毁家不汝相从。芜园归日正霜风。

繁华梦断，忍恨对残红。

挂角读书聊作计，何妨白眼鸡虫。人生多住大槐宫。

未应惆怅，草泽问鱼龙。
