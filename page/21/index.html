南歌子词二首 / 新添声杨柳枝词

发表于 2019-02-15 | 分类于 唐代
字数统计: 69 | 阅读时长 ≈ 1

作者：温庭筠
一尺深红胜曲尘，天生旧物不如新。(胜曲尘 一作：蒙曲尘) 合欢桃核终堪恨，里许元来别有人。 井底点灯深烛伊，共郎长行莫围棋。 玲珑骰子安红豆，入骨相思知不知。

南陵别儿童入京

发表于 2019-02-15 | 分类于 唐代
字数统计: 88 | 阅读时长 ≈ 1

作者：李白
白酒新熟山中归，黄鸡啄黍秋正肥。 呼童烹鸡酌白酒，儿女嬉笑牵人衣。 高歌取醉欲自慰，起舞落日争光辉。 游说万乘苦不早，著鞭跨马涉远道。 会稽愚妇轻买臣，余亦辞家西入秦。 仰天大笑出门去，我辈岂是蓬蒿人。

卜算子·咏梅

字数统计: 48 | 阅读时长 ≈ 1

作者：陆游
驿外断桥边，寂寞开无主。已是黄昏独自愁，更著风和雨。 无意苦争春，一任群芳妒。零落成泥碾作尘，只有香如故。

南风行十八日早作

发表于 2019-02-15 | 分类于 宋代
字数统计: 120 | 阅读时长 ≈ 1

作者：赵蕃
吾闻黄帝张乐洞庭野，始意庄生言故假。 又闻湘灵鼓瑟进玄螭，亦意屈子夸厥辞。 属行洞庭四月半，南风三日吹不断。 黄昏犹是天陆离，午夜忽成云片段。 病夫数起不知睡，数起侧听仍熟视。 恍疑帝乐尚铿鍧，又谓湘弦发幽閟。 乃知二子元非虚，吾闻自用耳目拘。 须臾月落天向晓，依旧南风波浩渺。

卜算子·我住长江头

发表于 2019-02-15 | 分类于 宋代
字数统计: 50 | 阅读时长 ≈ 1

作者：李之仪
我住长江头，君住长江尾。日日思君不见君，共饮长江水。 此水几时休，此恨何时已。只愿君心似我心，定不负相思意。

卜算子·黄州定慧院寓居作

发表于 2019-02-15 | 分类于 宋代
字数统计: 54 | 阅读时长 ≈ 1

作者：苏轼
缺月挂疏桐，漏断人初静。时见幽人独往来，缥缈孤鸿影。(时见 一作：谁见) 惊起却回头，有恨无人省。拣尽寒枝不肯栖，寂寞沙洲冷。
