巡视河防置酒晚归作二首

发表于 2017-02-23 | 分类于 宋代
字数统计: 44 | 阅读时长 ≈ 1

作者：宋祁
夹石双堤转，逍遥千骑留。
川昏多昼雾，水落足寒洲。
岸石危相倚，桥澌暮不流。
风烟非渭曲，谁有钓璜钩。

张公洞

发表于 2017-02-23 | 分类于 宋代
字数统计: 60 | 阅读时长 ≈ 1

作者：蔡肇
髯仙御气已超升，岩洞千秋冷不扃。
若许残铅除白发，敢辞长鑱斸青冥。
坛场岁为民祈福，蒿艾时因鬼乞灵。
他日重来寻石髓。绛云深坐读黄庭。

八月九日武林达宣差招宴时武良弼太守廉山御史同席

发表于 2017-02-23 | 分类于 元代
字数统计: 201 | 阅读时长 ≈ 1

作者：朱德润

八月九日天气凉，老仙会客登高堂。会稽太守廊庙器，黼黻政事多文章。

乌台御史守风节，州牧县令皆循良。盍簪偶尔动清赏，开筵四座罗壶觞。

太玄真人眉宇秀，辞辨析理推毫芒。举杯问答兴未已，歌舞落日增辉光。

娇鬟戏作猛将跃，大娘解演词人狂。冰弦戛击筝雁度，玉板拍裂鸣鼍轰。

婵娟鬓亸珠阿娜，细腰裙动金珰琅。霜糖炊粳荐膏膋，海错饤盘传蔗浆。

蜡炬照席飞银烬，桂花绕屋吹天香。人生得意须尽醉，酒渍啼痕秋梦长。

东山谢傅等陈迹，洛阳金谷何茫茫。月明上马清露下，无使乐极生悲伤。


待王十六监岳致语口号

发表于 2017-02-23 | 分类于 宋代
字数统计: 60 | 阅读时长 ≈ 1

作者：史浩

东皇旌旆到人寰，漏泄春工露一斑。已向江村访梅萼，却来灯市赏鳌山。

主宾冰玉欢无尽，伯季埙篪兴不悭。欲识鲁公须拜后，微黄一点见眉间。

卜算子慢 慰圭璋悼亡，依张子野体韵

发表于 2017-02-23 | 分类于 清代
字数统计: 97 | 阅读时长 ≈ 1

作者：汪东

鹍弦绝弄，鸾镜罢开，寂寂小庭春晚。摘尽残花，记否对花妆面。

留眄。滴梧桐、冷露如啼眼。怅旧日恩情，总逐浮萍浪蕊消散。

晓阜窥书馆。似鼓罢云和，曲终人远。宿草离离，已怕墓门青满。

谁遣。嗅余香、挂壁肠空断。待魂魄、经年重觅，证人天两见。
