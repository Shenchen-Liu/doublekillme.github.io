昼夜乐

发表于 2016-02-23 | 分类于 宋代
字数统计: 103 | 阅读时长 ≈ 1

作者：黄庭坚
夜深记得临岐语。说花时，归来去。教人每日思量，到处与谁分付。其奈冤家无定据。约云朝、又还雨暮。将泪入鸳衾，总不成行步。
元来也解知思虑。一封书、深相许。情知玉帐堪欢，为向金门进取。直待腰金拖紫后，有夫人、县君相与。争奈会分疏，没嫌伊门路。

宿茅山道院

字数统计: 61 | 阅读时长 ≈ 1

作者：皇甫汸

逶迤仙径礼神君，缥缈三峰锁白云。丹洞桃花春正发，星坛瑶草岁常芬。

山间鍊药人难识，天上吹笙世讵闻。独宿便思成道去，夜深鸾鹤幸来群。


书梅泉今觉盦诗集后二首 其一

发表于 2016-02-23 | 分类于 清代
字数统计: 61 | 阅读时长 ≈ 1

作者：陈曾寿

散原醲粹太夷巉，兼至重看今觉盦。振古才难天不惜，百年心死味犹谙。

有涯何苦殉哀乐，相望几时同笑谈。动地边风凄欲绝，尚吪无寐倚诗函。


买陂塘 吊明故宫

发表于 2016-02-23 | 分类于 魏晋
字数统计: 120 | 阅读时长 ≈ 1

作者：高燮

冷斜阳、南都旧事，伤心有恨难赋。乱砖片瓦皆尘梦，怪底残山如诉。

无证据。问宝座金銮，一例模糊处。龙蟠虎踞。剩故堞凄凉，任他毁卸，车载驴驮去。

荒芜久，不见衰烟寒树。谁怜宫殿狐兔。五龙桥上频回首，禁水无情暗度。

休恋顾。怕泪洒秋风，触物将倾注。沧桑计数。念历历兴亡，终成宿草，此理从今悟。

二月见梅花

发表于 2016-02-23 | 分类于 元代
字数统计: 61 | 阅读时长 ≈ 1

作者：赵秉文

不应开处避严冬，无赖春寒雪尚封。纵使风霜欺尔瘦，可能桃李为君容。

庞眉都尉嗟何晚，飞骑将军叹不逢。毕竟荣枯无定在，此花劝汝倒金钟。


广州吊古 其二

发表于 2016-02-23 | 分类于 清代
字数统计: 61 | 阅读时长 ≈ 1

作者：屈大均

白云山下即龙堆，旧日丁零大帐开。奇畜尽从沙漠至，远人兼带雪霜来。

无多越女留炎徼，不断明妃去紫台。朝汉有谁还朔望，雕青天子在蓬莱。
