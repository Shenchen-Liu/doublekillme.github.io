和武善夫韵二首 其二

发表于 2017-02-23 | 分类于 元代
字数统计: 62 | 阅读时长 ≈ 1

作者：耶律楚材

秋霖初霁觉新凉，午夜东山月吐光。翠竹无心甘晚节，黄花有意助年芳。

忠诚自许一心赤，老境谁怜两鬓霜。遥忆吾山归未得，故人书简怨东阳。


屏居

发表于 2017-02-23 | 分类于 宋代
字数统计: 45 | 阅读时长 ≈ 1

作者：释文珦
屏居泉石间，所乐在閒纵。
山田与山果，皆吾手自种。
禾穗已垂颖，栗包初拆缝。
一鲍既可期，肯复作诗用。

小重山 其二

发表于 2017-02-23 | 分类于 清代
字数统计: 63 | 阅读时长 ≈ 1

作者：毛奇龄

芳草茸茸翠辇迟。昭阳初日影、散彤墀。银幡綵胜戴双纚。

金错小，剪出万年枝。

御苑落花时。祠前祈百子、汉家池。承恩唯有臂红知。

中宵梦，频起视胭脂。


和杜中丞西禅院看花

发表于 2017-02-23 | 分类于 唐代
字数统计: 117 | 阅读时长 ≈ 1

作者：杨巨源
一林堆锦映千灯，照眼牵情欲不胜。知倚晴明娇自足，
解将颜色醉相仍。好风轻引香烟入，甘露才和粉艳凝。
深处最怜莺蹂践，懒时先被蝶侵凌。对持真境应无取，
分付空门又未能。迎日似翻红烧断，临流疑映绮霞层。
幽含晚态怜丹桂，盛续春光识紫藤。每到花枝独惆怅，
山东惟有杜中丞。

一剪梅 其三

发表于 2017-02-23 | 分类于 明代
字数统计: 64 | 阅读时长 ≈ 1

作者：俞彦

与君先后赋还山。我爱江山。君爱湖山。非关褊性合栖山。

叵奈冰山。转忆云山。

投琼佳句到青山。仰止高山。弦绝牙山。而今宿草北邙山。

梦破鱼山。泪洒牛山。

凄凄贱妾闭兰堂，渺渺征人戍朔方。漫拟龙城罢行役，空怜燕阁阅年芳。

和德清曹鲁郊七十一吟

发表于 2017-02-23 | 分类于 宋代
字数统计: 404 | 阅读时长 ≈ 1

作者：牟巘

元日书甲子，晓槃问五辛。颇记绛县老，宜有善颂陈。

君乎生是岁，早晚战伐尘。明年七十二，旧谍无纠纷。

斜飞滞栖枳，旁观叹积薪。欸乃来泽国，比屋多嚬呻。

何由跻寿域，君勿开荒榛。遗以大有年，在在鼓腹人。

非徒事勾稽，耄稚情日亲。溪流甚清泚，鹅山亦嶙峋。

胡为动乡思，便欲乞其身。驾言登东山，遐眺穷崖垠。

有时于沂浴，庶几远俗氛。平生布衣旧，方今鼎铉臣。

挽留不听去，好语温如春。悠然复少住，终惠此邑民。

九十以珍重，八十起渭滨。如君老益壮，眼明无眵昏。

精力殊未衰，齿宿意自新。盍陪英俊游，可是枯槁伦。

而况二三子，左右参鱼鳞。从翁取青紫，且姑置垂纶。

忆昨初相识，气象清以淳。而我已索如，发短不可耘。

蓬庐辱再访，顿觉双眉伸。大笑向我言，六甲那解轮。

惟须饮美酒，分饷良已勤。珍重君子意，自酌不及宾。

和气何盎盎，喜色何津津。长我梨与枣，平地即仙宸。

勿使他馈伤，小大相参辰。悛诵长短歌，尤足夷我神。

倘来视富贵，高情渺浮云。想其下笔时，健能斡千钧。

君倡我当和，撚髭檐屡巡。向来九老集，往往多沉沦。

亦欲相从去，病惫力弗臻。似闻纪初度，先我十二旬。

看君咏薰风，笑我醉霜旻。所期在千载，天地见本真。
